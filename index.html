Analysis of the Q Guide

The Harvard Q Guide has been scraped several times in the past, each time driving out certain elements of the data. Yet despite this, and despite the importance of understanding our own education, there is no one resource for understanding this data (besides the tools the University provides). In this analysis, I want to provide a starting point to do a lot of cool data analyses so that other people might be able to follow easily in my footsteps and create a shared collaborative space for Q Guide analysis.

There are so many natural questions that one might ask about the Q Guide itself.

  1. What is being taught and how has this changed over time?
  2. What are the best/hardest courses or concentrations?
  3. What is being said on the Q Guide and is there any way to systematically understand the comments?

We'll start with the information that everyone wants to see first: the workload and ratings scores. We weight by student not by class so as to not inflate the influence of small courses.
